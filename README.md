# IVF Embryo XAI Project

This repository contains the notebook used for experiments on IVF embryo selection with a Random Forest model and explainable AI methods (SHAP and LIME).

## Contents

- `ivf-embryo-xai-project.ipynb`: main notebook
  - Data loading and preprocessing
  - Model training and evaluation
  - Global and local explanations with SHAP and LIME

## How to run

1. Open the notebook in Google Colab or Jupyter.
2. Adjust the paths to the CSV files if needed.
3. Run all cells in order.
